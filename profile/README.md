## 🌌 About

**JustHappyLab** 是一个完全由兴趣驱动的开源实验室。  
不设边界，不被定义，想到什么就做什么。

这里没有固定赛道，只有持续冒出来的灵感、实验和作品。  
我们享受从 0 到 1 的过程，也热爱把有趣的想法真正做出来。
