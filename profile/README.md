## 🏠 关于我们

**JustHappyLab** — 一个完全由兴趣驱动的开源实验室。
不设边界，不被定义，想到什么就做什么。

- 📈 量化策略、交易框架、数据回测与分析
- 🎮 小游戏、独立 Demo、游戏工具与脚本
- 🎵 音乐制作辅助、音频处理、播放器工具
- 🔧 效率脚本、自动化工具、实用小软件
- 📱 iOS / Android 移动应用开发
- 🧩 微信小程序 / 轻应用开发
